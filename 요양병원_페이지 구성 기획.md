### **고객 페이지**



**메인페이지**

\-페이지 설명 노출 : 슬라이드로 광고 이미지 등록 및 요양 병원에 대한 긍정적 설명 노출

\-지도 노출 : 본인 위치+근처 병원 지도 입력

\-푸터: 회사 info, 문의 방법 info, contact info, 예금주 info, about us, 고객센터(문의하기 페이지)

**병원 설명 페이지**

\-정렬 : 가나다순, 혹은 입원 가능 총 인원수로 분류

\-신청란 : 설명 페이지 하단에 신청자명/환자명/환자와의관계/환자나이/주요병세/특이사항/전화번호/비밀번호/자동등록방지 신청하면 병원 관리 페이지에 데이터가 저장되며 문의 가능

**커뮤니티**

\-QNA:자주 들어오는 질문 페이지에서 해결이 되지 않는 질문을 문의하기 페이지로 이동 해결

\-상담 문의 페이지: 병원 설명 페이지 신청란과 연결되는 문의 페이지로 신청하고 나면 이곳에 비밀글로 등록됨

**병원 후기 페이지**

\-카테고리별 분류: 가입한 회원이 병원에 대한 향후 후기 및 정보를 적을 수 있는 페이지 (별점 및 이미지 첨부해 블로그형 후기 상세 기입 가능) 문제가 있는 글일 시 관리자 페이지에서 모니터링 후 삭제



### **관리 페이지**



**회원관리**

\-회원검색: 회원 전체 목록, 문자 및 이메일 발송 가능 회원 페이지 분류

\-휴면회원: 일정기간 미접속(약1\~2년) 인원 페이지 분류, 휴면 메일 자동 발송 기능, 관리자가 휴면 해제 가능하도록 하는 기능

\-탈퇴회원: 일정 기간이 지나지 않더라도 회원 완전삭제기능 추가, 완전삭제하지 않거나 일정 기간이 지나지 않을 시 데이터 보존.

**병원관리**

\-기본 병원 정보 등록: 병원 주소 및 수용 인원 등 기본 정보 작성

**환자관리**

\-문의환자목록 출력:등록일자로 분류 및 리스트 출력 기능

\-모집 공고 등록:관리 페이지에서 등록글 수정, 삭제 가능. 작성 기능을 관리자에만 넣는 것도 고려

**운영자게시판관리**

\-공지사항:공지글 작성 수정 삭제

\-QNA:자주하는 질문 작성 수정 삭제 및 이외 질문은 이메일으로 송신 받음
