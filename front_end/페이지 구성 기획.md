### **고객 페이지**



**메인페이지**

\-페이지 설명 노출 : 슬라이드로 광고 이미지 등록 및 요양 병원에 대한 긍정적 설명 노출

\-상품 노출 : 블록 형식으로 5개 당 1줄로 메인 카테고리 내 글 노출

\-푸터: 회사 info, 배송 방법 info, contact info, 예금주 info, about us, 고객센터(문의하기 페이지)

**가구 페이지**

\-세부카테고리 선택 페이지: 페이지로 이동 시 세부 카테고리 선택 가능, 선택 시 간편하게 볼 수 있도록 분류

\-상품상세페이지: 옵션 선택, 찜하기, 장바구니 추가, 바로 구매 기능

\-주문 및 결제 : 배송지 정보, 상품명, 상품번호, 적립금 및 쿠폰, 카드 결제 기능

**커뮤니티**

\-QNA:자주 들어오는 질문 페이지에서 해결이 되지 않는 질문을 문의하기 페이지로 이동 해결

**이벤트**

\-이벤트 페이지: 상품 중 제조사의 제안을 받아 선택된 일부 상품들이 홈페이지에 이벤트로 등록, 진행 중인 이벤트 및 지난 이벤트 카테고리를 분류해 두 가지로 나누기



### **관리 페이지**



**회원관리**

\-회원검색: 회원 전체 목록, 문자 및 이메일 발송 가능 회원 페이지 분류

\-휴면회원: 일정기간 미접속(약1\~2년) 인원 페이지 분류, 휴면 메일 자동 발송 기능, 관리자가 휴면 해제 가능하도록 하는 기능

\-탈퇴회원: 일정 기간이 지나지 않더라도 회원 완전삭제기능 추가, 완전삭제하지 않거나 일정 기간이 지나지 않을 시 데이터 보존.

**기업관리**

\-기본 회사 정보 등록: 반송지 등 기본 정보 작성

**상품관리**

\-상품목록 출력:등록일자/카테고리로 분류 및 리스트 출력 기능

\-상품 등록:관리 페이지에서 등록글 수정, 삭제 가능. 작성 기능을 관리자에만 넣는 것도 고려

**운영자게시판관리**

\-공지사항:공지글 작성 수정 삭제

\-QNA:자주하는 질문 작성 수정 삭제 및 이외 질문은 이메일으로 송신 받음

\-이벤트:진행 중 이벤트-지난 이벤트로 분류. 항상 최신 이벤트가 위에 오도록 배치. 작성 수정 삭제 가능



**매출관리**

통계:상품별, 일별로 카데고리 구분. 총매출/주문건수/배송비/수수료 정보 제공

&#x20;      ㄴ표 통계와 그래프 통계로 화면을 동시에 도출

통계-상세페이지: 주문번호 및 회원번호와 결제금액 확인 가능
